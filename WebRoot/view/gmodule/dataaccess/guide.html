数据访问: 数据访问对应数据库操作
访问URL: /dataaccess.json
参数说明: accessId:数据访问编号（必要参数）
map:其他参数（根据具体数据访问传入）
调用实例: $("#template_dg").datagrid({ url:basePath+"/dataaccess.json", queryParams:{ accessId:"3101" } });
实例说明: 通过编号为3101的数据访问，请求分页数据
返回值说明: 数据访问类型为分页查询，普通查询（多条记录），则返回形式{"rows":List<Map>}；
数据访问类型为普通查询（单条记录），则返回形式Map；
其他数据访问类型，返回形式{"resultCode":result}。
业务逻辑访问: 调用数据访问，处理数据访问结果
访问URL: /businesslogic.json
参数说明: logicId:业务逻辑访问编号（必要参数）
map:其他参数（根据具体业务逻辑访问传入）
调用实例: $.ajax({ url:basePath+"/businesslogic.json", data:{ logicId:"1000" }, success:function(){ } });
实例说明: 通过编号为1000的业务逻辑访问，请求数据
